how to install all requirement (windows):

1. initialize venv using this command `python -m venv .venv`
2. after it was installed run this script `.\venv\Script\activate.bat`
3. after it was succesfully activate the virtual environment
4. run this command to install all the script `pip install -r requirement.txt`
